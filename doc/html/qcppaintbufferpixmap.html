    QCPPaintBufferPixmap Class

    A paint buffer based on QPixmap, using software raster rendering. More...

    Header: #include <QCPPaintBufferPixmap>

    Detailed Description

    This paint buffer is the default and fall-back paint buffer which uses software rendering and QPixmap as internal buffer. It is used if \ref QCustomPlot::setOpenGl is false.
